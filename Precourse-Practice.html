What is HTML?

HTML is a markup language used to structure the content of our webpage. We can also say that HTML is a collection of elements which are expressed in the form of tags. Some of the elements in HTML are self-closing as it does not require closing tag.

Attribute is something which gives extra information about our HTML tag. We can write attribute in the opening tag of HTML (not closing tag). Attributes are expressed in the form of key-value pair which are differentiated by an equal-to sign. The value of Attribute is written between apostrophes.

CSS is used to style the element of our HTML. CSS can also be said to be collection of properties. Properties are also in the form of key-value pair but these pairs are differentiated using colon and ends with semi-colon.

Element Selector - Elements on which we need to apply CSS properties, we can select those elements using element selectors such as tags,ids and class. IDs always have a unique name.

CSS Properties.

color: This property is used to give color to a font.

height: This property is used to give height to our element.

width: This property is used to give width to our element.

margin: This property is used to give spacing between two different elements.

padding: This property is used to give spacing between the inner content of element and that element's border.

background-color: This property is used to give background color.

float: This property is used to specifies whether an element should float to the left, right, or not at all.

border: This property is used to border to our element, it has 3 values normally viz., width of border, type of border and color of border.

line-height: The line-height property specifies the height of a line.

box-shadow: The box-shadow property attaches one or more shadows to an element.

border-radius: The border-radius property defines the radius of the element's corners. This property allows you to add rounded corners to elements!

text-align: The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified.
